# agregador-imposto-de-renda-excel
Projeto desenvolvido na DIO para criação de um agregador de dados no Excel voltado à organização de informações para Declaração de Imposto de Renda, com validações, navegação intuitiva e consolidação automática.
Aplicar conceitos de organização e estruturação de dados no Excel
Criar uma ferramenta funcional com validações automáticas
Desenvolver uma interface amigável e intuitiva
Documentar o processo técnico de forma clara
